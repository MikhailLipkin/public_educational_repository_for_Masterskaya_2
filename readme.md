Этот репозиторий содержит следующие файлы:
    - Masterskaya_2.ipynb - файл, созданный с использованием языка Python в блокноте Jupyter с проектом, выполненным в спринте Мастерской_2 курса Data Scientist расширенный в Яндекс.Практикуме по созданию и обучению модели прогнозирования класса покупательской активности клиента интернет-магазина: проект включает в себя создание и обучение на тренировочной выборке моделей прогнозирования класса покупательской активности клиента интернет-магазина, тестирование этих моделей на данных Заказчика, выбор наилучшей модели и её анализ; файл может быть запущен и исполнен в любой интегрированной среде разработки (IDE), применимой к блокноту Jupyter;
    - requirements.txt - текстовый файл с описанием зависимостей проекта (перечислением все внешних Python-пакетов (библиотек), необходимых для работы проекта, для воспроизводства окружения, в котором проект гарантированно работает);
    - этот файл с описанием репозитория.
